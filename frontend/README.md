# Cafe Management☕♨️
Management app for cafe from which employee of your cofee shop can bill customers for the item. Also customers can choose and order their items from the screen placed in cafe.
#### Technologies
- React Js
- Node Js
- MongoDB
- Express
- Bootstrap
## Component
- Header
## To Run
To run the frontend part
```sh
cd frontend
npm install
npm start
```
To run the backend
```sh
npm install
nodemon app.js
```
